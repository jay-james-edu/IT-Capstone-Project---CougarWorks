# An "Easier" Degreeworks

A web application for students and advisors to track degree progress, built with Python and Django.

## Features

### For Students:
- View degree progress and remaining requirements
- Check GPA and academic standing
- Plan courses with "What-If" scenarios
- Track academic history

### For Advisors:
- Monitor assigned students' progress
- Add private notes to student profiles
- Receive alerts for academic standing changes
